Before You Come...

We want to make sure everyone who comes to NodeSchool gets the most out of it, so here are a few things to keep in mind:

  • If you've never done any code at all before, no HTML or CSS or Excel formulas or anything at all, the lessons we work on at NodeSchool are going to be pretty difficult. You should at least have a text editor and be able to open your command line tool. (If you're looking to get up to speed on the command line, read and work through this post.)
  • If you generally know how to fake your way through the command line but you need a laptop, let us know and we can provide one.
  • You should install node before you come. Make sure you know how to open your command line and run "node -v" to print the version of node you currently have. Also try "npm -v" to verify that npm was installed with node like it should have been.
  • Install the course you plan to work on during the event. Find instructions at nodeschool.io
